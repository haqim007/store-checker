# Store Checker
Visit stores or merchants

## Tech Stacks 
- Kotlin 
- Retrofit 
- Hilt 
- MVVM
- Coroutines
- Google Maps SDK
- Room

## Features 
- Show list store along with map with stores' tag in in
- Add tag to a store
- Set picture to a store
- Open direction in google maps app via this app
